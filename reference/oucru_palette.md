# Fetch OUCRU color palette by name (if it is defined)

Fetch OUCRU color palette by name (if it is defined)

## Usage

``` r
oucru_palette(palette = "main", ...)
```

## Arguments

- palette:

  Palette name

- ...:

  unused

## Value

OUCRU color palette

## Examples

``` r
# example code
oucru::oucru_palette("main")
#>     blue1    orange   yellow1   yellow2   yellow3 
#> "#002147" "#e08200" "#f5cf47" "#ffe3c4" "#fff7f0" 
```
